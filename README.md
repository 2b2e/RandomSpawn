### Random spawn plugin designed for Nukkit 1.0
* **Features**
- [x] Minimum & Max radius
- [x] /wild command
  - [ ] Cooldowns
  - **Permissions**
    - *rs.command.wild*: **Base access to the command.**
    - *rs.command.wild.bypass*: **Bypasses cooldowns and allows you to teleport otber players.**
- [ ] Preset spawns
